# Card-Simulator
抽卡模拟器

已知卡面分为r、sr、ssr、sp、ur类型，共有四个抽卡池，分为常驻池、活动sp卡池、活动ssr卡池、活动ur卡池。最好能动态调整每个卡池里面包含的卡。
1. 常驻池：卡池包含固定数量的r卡sr卡ssr卡和四张sp卡，r概率89% sr概率10% ssr&sp概率1%
2. 活动sp卡池：比常驻池多四张活动sp，所有ssr&sp概率2.5%，其中活动四张sp概率1.5%每张0.375%，r下调1.5%概率即为87.5%，sr概率与常驻池一致
3. 活动ssr卡池：比常驻池多四张活动ssr，所有ssr&sp概率2.5%，其中活动四张ssr概率1.5%每张0.375%，r下调1.5%概率即为87.5%，sr概率与常驻池一致
4. 活动ur卡池：比常驻池多四张活动ur，所有ssr&sp&ur概率2.5%，其中活动四张ur概率1.5%每张0.375%，r下调1.5%概率即为87.5%，sr概率与常驻池一致
所有卡池都遵循以下规则：
1. 卡池有单抽和十连抽两种模式
2. 十连抽的情况下，必定有一张sr


## 核心功能

✅ 历史记录管理
- 本地存储自动保存
- 支持删除单条记录
- 一键清空所有记录

✅ 灵活的筛选
- 按卡池类型筛选
- 按抽卡类型筛选（单抽/十连）
- 按稀有度筛选
- 组合筛选

✅ 详细的统计
- 总抽卡次数和卡牌数
- 各稀有度卡数和占比
- 保底触发率
- 稀有度分布图表

✅ 数据导出
- 导出JSON格式数据
- 支持浏览器下载

✅ 可视化展示
- 时间戳记录
- 卡池和类型标记
- 保底标记
- 进度条分布图
